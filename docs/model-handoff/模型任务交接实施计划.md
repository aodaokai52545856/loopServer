> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Tech Stack:** Markdown、PowerShell 7、Git CLI；不使用第三方 PowerShell 模块，不调用外部模型 API。

## 1. 唯一任务

`01-不可变架构决策.md` must record:

```markdown
# 不可变架构决策

## 四大组件与部署
服务器端是内网 GitLab 和 Loop Engine 控制面；个人电脑端是 repair-node 和 OpenCode。GitLab Runner 与 repair-executor 是 repair-node 安装包管理的内部执行部件。

## 权威边界
GitLab 管 Issue/仓库/Pipeline/Artifact/MR；控制面管解析、状态机、项目目录、节点、调度、审计和 Publisher；repair-node 管设备身份、心跳、能力、并发和 Runner；repair-executor 管单次执行；OpenCode 只修改工作区。

## 发布与凭据
个人节点没有仓库写 Token，只生成 Patch、manifest、事件和测试证据。只有服务器端 Publisher 可以推分支、commit 和创建 MR。OpenCode 启动前必须清除 GitLab 和任务凭据。

## 数据和幂等
PostgreSQL 是控制面权威状态；业务更新和 Outbox 同事务；Webhook、事件、reservation、分支和 MR 必须幂等；已提交 Flyway migration 不可修改。

## 固定状态
缺陷外部状态使用 `repair::new`、`repair::triaging`、`repair::needs-info`、`repair::queued`、`repair::running`、`repair::blocked`、`repair::failed`、`repair::ready-for-test`、`repair::cancelled`。

## 节点和调度
节点使用一次性邀请码和设备证书；15 秒心跳、45 秒离线；并发 1–10；先硬过滤再稳定排序；120 秒 reservation；模糊 Pipeline 结果必须对账，不能重复触发。

## 执行和测试
一期默认 OpenCode CLI JSON；每 Attempt 最多两轮测试反馈；事件先落本地 JSONL 再按 `attemptId + seq` 补传；验证命令不经过 shell。

## 待测试硬门槛
非空安全 Patch、必选测试通过、Artifact/manifest 校验、分支/commit/MR、完整 attempt/publish 事件全部存在，才可进入 `ready-for-test`。

## 技术和平台
Java 21、Spring Boot 4.1、PostgreSQL 17、Vue 3/TypeScript、Node 24 LTS、Go 1.26；Linux/macOS 正式支持，Windows Beta；一期不引入 Redis、NATS、Temporal、Kubernetes、MinIO。
```

[CmdletBinding()]
param([switch]$SkipIntegration)

    '03-启动提示词模板.md','04-完成报告模板.md','05-任务复核提示词.md','06-人工操作手册.md'
)
foreach ($name in $requiredDocs) {
    if (-not (Test-Path -LiteralPath (Join-Path $handoffRoot $name))) { throw "missing document: $name" }
}

$cards = @(Get-ChildItem -LiteralPath (Join-Path $handoffRoot 'task-cards') -Filter 'LE-P??-T??.md' | Sort-Object Name)
if ($cards.Count -ne 44) { throw "expected 44 cards, got $($cards.Count)" }
if (@($cards.BaseName | Sort-Object -Unique).Count -ne 44) { throw 'duplicate Task IDs' }
$expectedIds = @()
$phaseTaskCounts = @(7, 7, 8, 8, 7, 7)
for ($phase = 1; $phase -le 6; $phase++) {
    for ($task = 1; $task -le $phaseTaskCounts[$phase - 1]; $task++) {
        $expectedIds += 'LE-P{0:D2}-T{1:D2}' -f $phase, $task
    }
}
if (@(Compare-Object $expectedIds @($cards.BaseName)).Count -ne 0) { throw 'Task ID set is not the required 44-ID sequence' }

$requiredSections = @('唯一目标','明确不做','前置条件','开始状态','必须阅读','允许修改','禁止修改','不可变约束','执行顺序','验收命令','停止条件','完成报告')
$index = Get-Content -Raw -LiteralPath (Join-Path $handoffRoot '02-任务索引.md')
if (([regex]::Matches($index, '(?m)^\|.*\| READY \|\r?$')).Count -ne 1) { throw 'index must contain exactly one READY row' }
if (([regex]::Matches($index, '(?m)^\|.*\| PENDING \|\r?$')).Count -ne 43) { throw 'index must contain exactly 43 PENDING rows' }
for ($cardIndex = 0; $cardIndex -lt $cards.Count; $cardIndex++) {
    $card = $cards[$cardIndex]
    $text = Get-Content -Raw -LiteralPath $card.FullName
    if (-not $text.StartsWith("# $($card.BaseName) —")) { throw "ID/title mismatch: $($card.Name)" }
    if ($text -match 'TBD|TODO|\{[A-Z_]+\}|matching test|参见|同上') { throw "placeholder or shorthand: $($card.Name)" }
    foreach ($section in $requiredSections) {
        if (-not $text.Contains("## $section")) { throw "$($card.Name) missing $section" }
    }
    if ($text -notmatch '(?ms)^## 允许修改\r?\n- `') { throw "$($card.Name) has no concrete allowlist" }
    $expectedPrerequisite = if ($cardIndex -eq 0) { '无' } else { $cards[$cardIndex - 1].BaseName }
    $prerequisite = [regex]::Match($text, '(?m)^\*\*前置任务：\*\* (.+)$').Groups[1].Value.Trim()
    if ($prerequisite -ne $expectedPrerequisite) { throw "$($card.Name) prerequisite is $prerequisite, expected $expectedPrerequisite" }
    $planPath = [regex]::Match($text, '\*\*实施计划：\*\* `([^`]+)`').Groups[1].Value
    $heading = [regex]::Match($text, '\*\*计划任务标题：\*\* `([^`]+)`').Groups[1].Value
    if ([string]::IsNullOrWhiteSpace($planPath) -or [string]::IsNullOrWhiteSpace($heading)) { throw "$($card.Name) missing plan metadata" }
    $resolvedPlan = [IO.Path]::GetFullPath((Join-Path $card.DirectoryName $planPath))
    if (-not (Test-Path -LiteralPath $resolvedPlan)) { throw "$($card.Name) references missing plan" }
    $phaseNumber = $card.BaseName.Substring(4, 2)
    $taskNumber = [int]$card.BaseName.Substring(8, 2)
    if (-not ([IO.Path]::GetFileName($resolvedPlan)).StartsWith("实施计划-$phaseNumber-")) { throw "$($card.Name) references the wrong phase plan" }
    if (-not (Select-String -Quiet -LiteralPath $resolvedPlan -SimpleMatch "### Task ${taskNumber}: $heading")) { throw "$($card.Name) references missing exact heading" }
    $expectedLink = "[$($card.BaseName)](./task-cards/$($card.Name))"
    if (-not $index.Contains($expectedLink)) { throw "index missing link $expectedLink" }
}
